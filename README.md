# Iris
Iris-ROS-Lite: A Python-based, multi-threaded HRI stack for Poppy Robots. > Decoupled architecture using a lightweight internal event bus. Includes real-time viseme lip-sync, Pygame-based facial expressions, MediaPipe gesture recognition, and pluggable LLM backends (Groq/Ollama/Gemini). Low-latency autonomy for the Raspberry Pi 5.
